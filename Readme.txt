Challenger Conversor de moneda utilizando api

Esta es la segunda versión del conversor de moneda, el cual utilice como referencia otros proyectos enviados al discord de alura.
junto con la referencia de foros, YouTube, las clases de alura, entre otros.
El mayor punto que me trajo problemas fue la conexión de la api con java, la mayor parte de esta fue utilizada de otro proyecto.
En versiones posteriores pretendo utilizar otro método distinto para aplicar el api.

Descripción del proyecto:
El proyecto se separa principalmente en 4 clases diferentes.
La clase principal, donde existe un main, que es la interacción del usuario con la aplicación.
La clase consultaApi, la cual seria la conexión del api con java.
La clase divisaOmdb, el cual es una clase record, para obtener todos los get y set de la api.
La clase GeneradorDeArchivos, el cual es una clase que contiene la programación para generar el archivo para ir obteniendo los diferentes precios de las 
monedas actuales.
